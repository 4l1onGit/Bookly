# Symfony Powered fullstack Book Review Site

A fullstack application powered by Symfony to allow users to review books and add books. This version includes a RESTful api that can be accessed using curl/postman which is protected using JWT authentication.

- Symfony backend
- MVC
- Twig for templating
- Doctrine ORM for database interactions
- Bootstrap 5 for styling
- CRUD operations

### Assignment 2 Related Features ###
  
- RESTful API
- JWT Auth
- Guzzle (Recieve data from two external APIs)
